# 2. 프로젝트 개요

- 프로젝트 명 : 전국 전기자동차 등록 현황 및 FAQ 조회시스템 

- 프로젝트 소개 : 전기자동차 등록 현황, 전기자동차 차량 정보 및 구매보조금 확인, 전기차 충전소 위치기반, 전기차 FAQ

- 프로젝트 필요성(배경)
  친환경 전기차 이용률이 늘어남에 따라 전기차 관련 데이터를 제공함으로써 전기차 구매 및 사용에 도움을 제공하고자 함

- 프로젝트 목표
 데이터 크롤링 및 STREAMLIT을 활용한 데이터 제공
